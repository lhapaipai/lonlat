## Installation

dans le dossier `apps/front`
```bash
pnpm install
```

## Démarrage

Front
```bash
cd apps/front
pnpm dev
```

## Prérequis

### Nodejs

On passera par volta pour installer node (ce qui nous permet d'utiliser différentes version de node sur un même système)

```bash
# install Volta
curl https://get.volta.sh | bash

# install Node
volta install node@20

# start using Node
node --version
```
